# Task7_Logistic_Regression_Titanic_Survival_prediction


## Objective
Predict whether a passenger survived the Titanic disaster using Logistic Regression.

## Dataset
Titanic dataset from Kaggle / Seaborn.

## Steps Performed
- Data cleaning & missing value handling
- Encoding categorical features
- Feature scaling
- Logistic Regression model training
- Model evaluation using accuracy, precision, recall, F1-score
- ROC curve and AUC score analysis

## Results
The model achieved good classification performance with a reliable AUC score.

## Tools Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
